기존에진행했던PoC(Proof of Concept) 코드의 구조와 로직을 참고하여,
데이터를JSON 파일로 관리하는CRUD(Create, Read, Update, Delete) 콘솔 애플리케이션을 개
발한다

요구사항
• PoC에서 사용된코드구조를유지한상태로, CRUD 구현
CRUD 기능 구현
• Create: 새로운 데이터를 입력 받아JSON 파일에 저장.
• Read: 전체 목록 보기및특정ID/키값으로검색기능.
• Update: 기존 데이터를 선택하여특정필드수정.
• Delete: 특정 데이터를 안전하게삭제
