# crud-25021572

데이터를 JSON 파일로 관리하는 CRUD(Create, Read, Update, Delete) 콘솔 애플리케이션입니다.

구현 세부 내용은 [Implementation.md](Implementation.md)를 참고하세요.

## 실행

```bash
python main.py
```

## 메뉴

```text
==== CRUD 콘솔 애플리케이션 ====
1. Create
2. Read (전체 목록)
3. Read (검색)
4. Update
5. Delete
0. 종료
```

번호를 입력해 원하는 기능을 선택합니다.

- **1. Create**: `이름`, `값`을 순서대로 입력하면 새 항목이 `id`와 함께 생성되어 `data.json`에 저장됩니다.
- **2. Read (전체 목록)**: 저장된 모든 항목을 출력합니다. 데이터가 없으면 안내 메시지를 출력합니다.
- **3. Read (검색)**: `id` 또는 `이름`을 입력하면 해당 항목을 찾아 출력합니다. 숫자를 입력하면 id로, 그 외에는 이름으로 검색합니다.
- **4. Update**: `id` 또는 `이름`으로 대상을 찾은 뒤, 수정할 필드명(`name` 또는 `value`)과 새 값을 입력하면 반영됩니다.
- **5. Delete**: `id` 또는 `이름`으로 대상을 찾은 뒤 `y`를 입력하면 삭제됩니다. `y`가 아닌 값을 입력하면 삭제가 취소됩니다.
- **0. 종료**: 프로그램을 종료합니다.

## 예시

```text
선택: 1
이름: 홍길동
값: 30
생성 완료 (id=1)

선택: 3
검색할 ID 또는 이름: 홍길동
{'id': 1, 'name': '홍길동', 'value': '30'}

선택: 4
수정할 ID 또는 이름: 1
수정할 필드 (name, value): value
새 값: 35
수정 완료

선택: 5
삭제할 ID 또는 이름: 1
정말 삭제하시겠습니까? {'id': 1, 'name': '홍길동', 'value': '35'} (y/n): y
삭제 완료
```
